My Blind Spots 2018

The project:

My Blind Spots is a static client site that helps people discover their behaviours and build strong relationships with others.

Designs & challenges:

To maintain sharpness like an SVG image would, I've decided to use clipaths for the oval cutouts on the banners throughout the site. For the most part, this works, however, clipspaths work best in relative units (i.e percentage), therefore the wideness and curveness will not be consistent with all screen sizes.

Repositioning and resizing of the puzzle background image on different viewports.

Boxti 2018

The project:

Boxti is an e-commerce site that offers all sorts of bundled gifts for any occasion.

Designs & challenges:

This site is powered by the Woo Commerce plugin on Wordpress. Heavy css customisation is applied on top of the default styles to achieve the custom approved design by the client. This includes css and jquery manipulation, and especially tricky when elements are deeply nested and not classified.

jQuery methods such as .parent(), .find(), and css selectors .nth-child() made targeting nested elements easier.

Kader Boot Co. 2018

The project:

Kader Boot Co. is an e-commerce store that sells unique Australian designed boots.

Designs & challenges:

The round toe/square toe boot's background image position and size had to change when the viewport changes. This was challenging because I had to fit some text without overlapping the background image.

The product filter on the shop page is a Wordpress plugin. The default styles are just generic inputs and labels. I achieved the new design by using css's flexbox, before/after pseudos, and background images for the boot icons.
